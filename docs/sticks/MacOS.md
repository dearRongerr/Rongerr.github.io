# MacOS

## 终端命令

-  `cd ..`  返回上级文件
- `ls`  显示当前目录文件
- `ls -a` 显示当前目录的所有文件，包括隐藏文件

**macOS 终端命令**

- 显示文件树

  - 打开mac终端

    - 输入 brew install tree

    - 使用：

      `tree` 显示文件树

      `tree -a` 显示所有文件树，包含隐藏文件
